## Introducción

El componente `EventsList` muestra los eventos disponibles en una grilla responsive.

<Info>
Ver documentación completa en [Gestión de Eventos](/features/events).
</Info>

## Código Fuente

El código fuente está disponible en:
- `src/components/apps/events/EventsList.js`
- `src/views/events/Events.js`

## Características

- Grid responsive (2-4 columnas según pantalla)
- Carga automática desde Firestore
- Navegación a checkout al seleccionar
- Imágenes con hover effect
