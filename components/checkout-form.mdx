## Introducción

El formulario de checkout maneja toda la información del cliente y procesamiento de pagos.

<Info>
Ver documentación completa en [Proceso de Checkout](/features/checkout).
</Info>

## Código Fuente

El código fuente está disponible en:
- `src/views/checkout/CheckOut.js`

## Características

- Validación con Formik + Yup
- Soporte para clientes corporativos
- Múltiples métodos de pago
- Cálculo automático de impuestos
- Temporizador de reserva
