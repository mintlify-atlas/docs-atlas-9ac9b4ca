## Introducción

El módulo de entradas de cortesía permite a los operadores de taquilla emitir tickets gratuitos o promocionales para eventos. Este sistema tiene su propio flujo independiente del checkout regular.

<Note>
Las entradas de cortesía no requieren procesamiento de pago pero mantienen el registro completo del beneficiario en la base de datos.
</Note>

## Acceso al Módulo

El módulo de cortesía está protegido por `OfficeGuard` y requiere autenticación:

## Flujo de Cortesía

<Steps>
  <Step title="Selección de Evento">
    El operador selecciona el evento para el cual emitir cortesías desde `/cortesia`
  </Step>
  
  <Step title="Proceso de Emisión">
    En `/cortesia-proceso`, el operador:
    - Selecciona asientos/zonas
    - Ingresa datos del beneficiario
    - Especifica motivo de la cortesía
  </Step>
  
  <Step title="Confirmación">
    En `/cortesia-resumen`, se muestra el resumen y se genera el ticket
  </Step>
</Steps>

## Componente: CourtesyTicketsList

Muestra los eventos disponibles para emitir cortesías:

```javascript src/components/apps/courtesy/CourtesyTicketsList.js
import { collection, getDocs } from 'firebase/firestore';
import { getEventData } from '../../../store/apps/event/eventSlice';

const CourtesyTicketsList = () => {
    const [events, setEvents] = useState([]);
    const [loading, setLoading] = useState(true);
    const OfficeAuth = useSelector((state) => state.office.office);
    const dispatch = useDispatch();
    const navigate = useNavigate();

    const goToCourtesyProcess = (eventData) => {
        dispatch(getEventData({ event: eventData }));
        navigate('/cortesia-proceso');
    };

    useEffect(() => {
        const fetchData = async () => {
            setLoading(true);
            try {
                // Verificar autenticación de taquilla
                if (!OfficeAuth?.id) {
                    toast.error('No hay información de oficina disponible');
                    return;
                }

                const querySnapshot = await getDocs(
                    collection(db, `offices/${OfficeAuth.id}/events`)
                );

                if (!querySnapshot.empty) {
                    const events = querySnapshot.docs.map(doc => ({
                        id: doc.id,
                        ...doc.data()
                    }));
                    setEvents(events);
                }
            } catch (error) {
                console.error("Error al cargar eventos:", error);
                toast.error('Error al cargar eventos');
            } finally {
                setLoading(false);
            }
        };

        fetchData();
    }, [OfficeAuth]);

    return (
        <ul className="grid grid-cols-2 gap-4">
            {events.map((event) => (
                <li key={event.id}>
                    <div className="event-card">
                        <img src={event?.event.media[0]} alt="" />
                        <button onClick={() => goToCourtesyProcess(event)}>
                            Gestionar cortesías
                        </button>
                    </div>
                    <p>{event?.event.name}</p>
                </li>
            ))}
        </ul>
    );
};
```

## Validaciones Especiales

<Warning>
Las entradas de cortesía están sujetas a límites configurables por evento para evitar abusos.
</Warning>

```javascript
// Ejemplo de validación
const validateCourtesyLimit = async (eventId) => {
  const eventDoc = await getDoc(doc(db, 'events', eventId));
  const courtesyConfig = eventDoc.data().courtesy_config;
  
  if (!courtesyConfig.enabled) {
    throw new Error('Las cortesías no están habilitadas para este evento');
  }
  
  const courtesyCount = await getCourtesyCount(eventId);
  
  if (courtesyCount >= courtesyConfig.max_tickets) {
    throw new Error(`Se alcanzó el límite de ${courtesyConfig.max_tickets} cortesías`);
  }
  
  return true;
};
```

// Verificar permisos para emitir cortesías
const canIssueCourtesy = (user, officeAuth) => {
  // Solo supervisores y administradores pueden emitir cortesías
  return user.role === 'supervisor' || user.role === 'admin';
};

// Verificar límite diario por operador
const checkDailyLimit = async (userId) => {
  const today = new Date().toISOString().split('T')[0];
  const courtesiesToday = await getCourtesyCountByUser(userId, today);
  
  return courtesiesToday < MAX_COURTESY_PER_DAY;
};
