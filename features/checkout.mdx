## Introducción

El proceso de checkout de TMT es un flujo de dos pasos diseñado para garantizar una experiencia de compra fluida y segura. Combina selección de asientos, validación de datos del cliente, múltiples métodos de pago y cálculo automático de impuestos.

## Flujo de Checkout en Dos Pasos

<Steps>
  <Step title="Paso 1: Selección de Asientos">
    - Visualización de asientos disponibles (manual o Seats.io)
    - Selección múltiple de asientos
    - Cálculo de subtotal en tiempo real
    - Vista previa de asientos seleccionados
  </Step>
  
  <Step title="Paso 2: Información de Pago">
    - Datos del cliente (nombre, email, teléfono, ID)
    - Selección de métodos de pago
    - Cálculo de impuestos (IVA, IGTF)
    - Confirmación y procesamiento
    - Reserva de asientos (10 minutos)
  </Step>
</Steps>

<Warning>
Los asientos se reservan por **10 minutos** (600 segundos) al entrar al Paso 2. Si el tiempo expira, los asientos se liberan automáticamente.
</Warning>

## Estado del Checkout

El componente `CheckOut` maneja múltiples estados para gestionar el flujo:

```javascript src/views/checkout/CheckOut.js
const CheckOut = () => {
```

## Paso 1: Selección de Asientos

### Sistemas de Selección

TMT soporta dos sistemas de selección de asientos:

<Tabs>
  <Tab title="Sistema Manual">
    Sistema de grilla con botones clickeables. Los asientos se cargan desde Firestore:
    
    ```javascript
    const generateSeats = (zone, tickets) => {
      const sortedTickets = tickets.sort((a, b) => {
        const numA = parseInt(a.seat_id.match(/\d+/)[0], 10);
        const numB = parseInt(b.seat_id.match(/\d+/)[0], 10);
        return numA - numB;
      });

      return sortedTickets.map((ticket) => ({
        id: ticket.seat_id,
        ticket_id: ticket.ticket_id,
        price: zone.price,
        corporate_price: zone.corporate_price,
        status: ticket.status === true ? 'available' : 'unavailable',
        location: zone.name,
      }));
    };
    ```
  </Tab>
  
  <Tab title="Seats.io">
    Mapa interactivo con selección visual. Se detecta automáticamente:
    
    ```javascript
    const eventDoc = await getDoc(doc(db, 'events', eventData?.id));
    
    if (eventDoc.exists()) {
      const eventDataFromDb = eventDoc.data();
      if (eventDataFromDb.seatsio_event?.chartKey) {
        setHasSeatsioMap(true);
        setSeatsioEventData(eventDataFromDb.seatsio_event);
      }
    }
    ```
  </Tab>
</Tabs>

### Cálculo de Precios

Los precios varían según el tipo de cliente:

```javascript
const calculateSubtotal = () => {
  const subtotal = selectedSeats.reduce((sum, seat) => {
    const price = clientType === 'corporativo' 
      ? (seat.corporate_price || seat.price) 
      : seat.price;
    return sum + (price || 0);
  }, 0);
  return truncateDecimals(subtotal, 2);
};
```

<Info>
Los precios corporativos se aplican automáticamente cuando `clientType === 'corporativo'` y el asiento tiene `corporate_price` definido.
</Info>

## Paso 2: Información de Pago

### Transición al Paso 2

Al confirmar los asientos, el sistema realiza varias acciones:

```javascript
const goToStep2 = async () => {
  setSubmitConfirmSeat(true);
  dispatch(setLoadingData({ loading: true }));
  dispatch(fetchSetup()); // Cargar configuración global

  // Reservar asientos por 10 minutos
  const seatsArray = selectedSeats.map((seat) => seat.ticket_id);
  const lockSeats = await tickets_lock({ 
    ticketd_reserved: seatsArray, 
    reserved_time: 600 
  });

  if (lockSeats.data.valido) {
    setStep(2);
    resetStep2Fields();
    setIsTimerRunning(true); // Iniciar temporizador
  } else {
    // Mostrar asientos no disponibles
    lockSeats.data.nodisponible.forEach((id) => {
      const ticketsFilter = selectedSeats.filter((x) => x.ticket_id === id);
      ticketsFilter.forEach((tickets) => {
        toast.error(`El ticket ${tickets.id} no está disponible`);
      });
    });
  }
};
```

<Warning>
Si algún asiento no está disponible al intentar reservar, el usuario permanece en el Paso 1 y debe seleccionar asientos diferentes.
</Warning>

### Temporizador de Reserva

El temporizador visual muestra el tiempo restante:

```javascript
useEffect(() => {
  let timer;
  if (isTimerRunning && timeLeft > 0) {
    timer = setTimeout(() => setTimeLeft(timeLeft - 1), 1000);
  } else if (timeLeft === 0) {
    setShowTimeoutDialog(true);
    setIsTimerRunning(false);
  }
  return () => clearTimeout(timer);
}, [isTimerRunning, timeLeft]);
```

El formulario utiliza Formik + Yup para validación:

```javascript
const OrderSchema = Yup.object().shape({

  customer_phone: Yup.string()
    .required("Se requiere el Teléfono del cliente"),
  customer_id: Yup.string()
    .required("Se requiere la Cédula de Identidad del cliente"),
  customer_id_type: Yup.string()
    .required("Se requiere el tipo de documento"),
  customer_address: Yup.string()
    .required("Se requiere la Dirección del cliente"),
  customer_country: Yup.string()
    .required("Se requiere el País del cliente"),
  // Campos adicionales para clientes corporativos
  ...(clientType === 'corporativo' && {

    company_rif: Yup.string()
```

## Procesamiento de la Orden

Cuando el usuario confirma la compra:

<Steps>
  <Step title="Cálculo de Totales">
    ```javascript
    const subtotal = calculateSubtotal();
    const taxes = calculateTaxes(subtotal); // IVA 16%
    const igtf = calculateIGTF(); // IGTF 3% si hay USD
    const total = subtotal + taxes + igtf;
    ```
  </Step>
  
  <Step title="Preparar Transacciones">
    ```javascript
    let transactions = await Promise.all(filteredPayments.map(async (payment) => {
      const rawAmount = Number(paymentAmounts[payment.payment_method_id]);
      const amountInUsd = payment.currency === 'USD' 
        ? rawAmount 
        : rawAmount / usdCentralBank;
      
      return {
        amount: rawAmount,
        amount_currency: payment.currency,
        amount_exchange: amountInUsd,
        amount_exchange_rate: payment.currency === 'USD' ? 1 : usdCentralBank,
        date: serverTimestamp(),
        payment_id: payment.payment_method_id,
        payment_name: payment.name,
        status: true,
      };
    }));
    ```
  </Step>
  
  <Step title="Crear Orden">
    ```javascript
    const finalData = {
      amount: finalTotalAmount,
      event_id: eventData.id,
      event_name: eventData.event.name,
      office_id: officeData.id,
      tickets: tickets,
      transactions: transactions,
      exchange_rate: usdCentralBank,
      is_corporate: clientType === 'corporativo'
    };
    
    const resp = await order_created(finalData);
    ```
  </Step>
  
  <Step title="Confirmar Asientos en Seats.io">
    ```javascript
    if (hasSeatsioMap && seatsioHoldToken) {
      const seatsToBook = prepareSeatsForSeatsio(selectedSeats);
      await book_seats_seatsio({
        event_id: seatsioEventData.eventKey,
        seats: seatsToBook,
        hold_token: seatsioHoldToken
      });
    }
    ```
  </Step>
  
  <Step title="Enviar Confirmación por Email">
    ```javascript
    send_email({
      to_email: customerEmail,
      to_name: customerName,
      html_send: htmlContent,
      subject: `Confirmación de compra: ${eventData.event.name}`
    });
    ```
  </Step>
</Steps>
