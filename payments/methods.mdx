## Métodos de Pago Disponibles

TMT soporta múltiples métodos de pago que se cargan dinámicamente desde la configuración de cada taquilla. Los métodos se pueden combinar en una misma transacción.

```javascript CheckOut.js (líneas 78-87)
const querySnapshot2 = await getDocs(collection(db, `/offices/${officeData?.id}/payment_methods`));
let PaymentMethodsFinal = [];
let PaymentMethodsFinal2 = [];
querySnapshot2.forEach((resp) => {
  const data = resp.data();
  PaymentMethodsFinal.push({ label: data.name, value: data.method, id: data.payment_method_id });
  PaymentMethodsFinal2.push(data);
});
setPaymentMethods(PaymentMethodsFinal);
setPaymentMethodsData(PaymentMethodsFinal2);
```

<Note>
  Cada taquilla puede tener configurados diferentes métodos de pago según sus necesidades operativas.
</Note>

### Descripción
Pago internacional vía Zelle para transacciones en dólares estadounidenses.

</ParamField>

## 4. Tarjetas de Crédito y Débito

### Descripción
Pagos con tarjetas de crédito o débito procesados mediante puntos de venta.

### Campos Requeridos

<ParamField path="punto_venta" type="string" required>
  Punto de venta utilizado: "tmt" o "productora"
</ParamField>

<ParamField path="reference_number" type="string" required>
  Número de referencia de la transacción del punto de venta
</ParamField>

### Sistema de Puntos de Venta

Las tarjetas pueden procesarse mediante dos tipos de puntos de venta:

<Accordion title="TMT (por defecto)">
  Punto de venta propio de TMT. Se marca automáticamente con `point_sale_tmt: true`.
</Accordion>

<Accordion title="Productora">
  Punto de venta de la productora del evento. Se marca con `point_sale_tmt: false`.
</Accordion>

```javascript CheckOut.js (líneas 275-278)
else if (payments.method === "Tarjeta de Crédito" || payments.method === "Tarjeta de Debito") {
```

## Pagos Mixtos

El sistema permite combinar múltiples métodos de pago en una sola transacción:

```javascript CheckOut.js (líneas 1201-1224)
const togglePaymentMethod = (method) => {
  const methodId = method.id;
  const isCurrentlySelected = selectedPaymentMethods.includes(methodId);

  if (isCurrentlySelected) {
    // Deselect it
    setSelectedPaymentMethods(selectedPaymentMethods.filter(id => id !== methodId));
    // Remove amount
    setPaymentAmounts(currentAmounts => {
      const newAmounts = { ...currentAmounts };
      delete newAmounts[methodId];
      return newAmounts;
    });
    // Remove additional data
    setAdditionalPaymentData(prevData => {
      const newData = { ...prevData };
      delete newData[methodId];
      return newData;
    });
  } else {
    // Select it
    setSelectedPaymentMethods([...selectedPaymentMethods, methodId]);
  }
};
```

### Cálculo de Montos Totales

```javascript CheckOut.js (líneas 1141-1157)
const getTotalEnteredAmount = () => {
  let totalInUSD = 0;
  for (const methodId in paymentAmounts) {
    const amount = parseFloat(paymentAmounts[methodId] || 0);
    if (amount > 0) {
      const methodData = paymentMethodsData.find(p => p.payment_method_id === methodId);
      if (methodData) {
        if (methodData.currency === 'USD') {
          totalInUSD += amount;
        } else {
          totalInUSD += (amount / usdCentralBank);
        }
      }
    }
  }
  return totalInUSD;
};
```

<Warning>
  El sistema valida que la suma de todos los métodos de pago coincida exactamente con el total a pagar (incluyendo impuestos). Se permite una diferencia máxima de 0.01 USD.
</Warning>

## Cuentas de Custodia

Algunos métodos de pago pueden tener cuentas de custodia asociadas:

```javascript CheckOut.js (líneas 282-298)
if (payments.custody_account?.id && payments.custody_account?.id.id !== '') {
  try {
    const custodyAccountDoc = await getDoc(doc(db, `custody_accounts/${payments.custody_account?.id}`));
    if (custodyAccountDoc.exists()) {
      const accountData = custodyAccountDoc.data();
      payments.custody_account = {
        ...payments.custody_account,
        account_number: accountData.metadata?.account_number || ''
      };
    }
  } catch (error) {
    console.error("Error al obtener cuenta custodia:", error);
  }
}
```

## Validación Completa

La función `arePaymentMethodFieldsComplete` valida todos los campos requeridos:

```javascript CheckOut.js (líneas 1164-1199)
const arePaymentMethodFieldsComplete = () => {
  for (const methodId of selectedPaymentMethods) {
    const methodInfo = paymentMethodsData.find(m => m.payment_method_id === methodId);
    if (!methodInfo) continue;

    const methodData = additionalPaymentData[methodId] || {};

    // Validaciones específicas según el tipo de método
    switch (methodInfo.method) {
      case 'Zelle':
        if (!methodData.email || !methodData.name || !methodData.phone) {
          return false;
        }
        break;
      case 'Pago Móvil':
        if (!methodData.reference_number || !methodData.phone || !methodData.bank) {
          return false;
        }
        break;
      case 'Transferencia Bancaria Nacional':
        if (!methodData.reference_number || !methodData.account_number || !methodData.bank) {
          return false;
        }
        break;
      case 'Tarjeta de Crédito':
      case 'Tarjeta de Debito':
        if (!methodData.reference_number) {
          return false;
        }
        break;
      // Para otros métodos, no requerimos campos adicionales
    }
  }

  return true;
};
```
