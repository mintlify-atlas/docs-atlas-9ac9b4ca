## Qué es TMT - Taquilla Física

TMT es un sistema de punto de venta especializado para taquillas físicas de eventos. Permite a los operadores de taquilla vender entradas de manera eficiente, gestionar múltiples métodos de pago, y ofrecer una experiencia fluida tanto para eventos con asientos numerados como con selección libre.

## Comenzar es Fácil

<Steps>
  <Step title="Instalación">
    Clona el repositorio y configura las dependencias
    
    ```bash
    git clone <repository-url>
    cd tmt-taquilla-fisica
    npm install
    ```
  </Step>
  
  <Step title="Configuración">
    Configura Firebase y las variables de entorno necesarias
    
    ```bash
    # Copia el archivo de configuración
    cp .env.example .env
    
    # Edita .env con tus credenciales
    ```
  </Step>
  
  <Step title="Desarrollo">
    Inicia el servidor de desarrollo
    
    ```bash
    npm run dev
    ```
  </Step>
</Steps>

## Tecnologías Utilizadas

TMT está construido con tecnologías modernas y confiables:

- **React 18** - Framework de interfaz de usuario
- **Vite** - Build tool ultrarrápido
- **Redux Toolkit** - Gestión de estado global
- **Firebase** - Backend as a Service
- **Tailwind CSS** - Framework de estilos utilitarios
- **PrimeReact** - Componentes UI avanzados
- **Formik + Yup** - Gestión y validación de formularios
- **Seats.io** - Selección de asientos interactiva
